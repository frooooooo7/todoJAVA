## Getting Started

### Prerequisites

- Java JDK 11+
- Maven 3.6+

### Installation

1. Clone the repository:
   
   ```
   git clone https://github.com/frooooooo07/todoJAVA
   
   ```
2. Navigate to project directory:
   
   ```
   cd todoJAVA
   
   ```
3. Install dependencies:
   
   ```
   mvn install
   
   ```

### Usage

Run the application:
  ```
  mvn exec:java

  ```
